How to use

HTML

<div id="epcounter"> <div>An expire: <span id="expireMount"></span> <span id="expireDays"></span></div> <div>A days left: <span id="daysLeft"></span></div> </div>

Base initializing

$("#epcounter").epcounter({ "idExpireMonth": "expireMount", "idExpireDays": "expireDays", "idDaysLeft": "daysLeft" });

Set a start date and amount promotion days

$("#epcounter").epcounter({ "amountPromotionDays": 15, "startDay": 1, // Day of month 1 - 31 "startMonth": 8, // Month 1 - 12, "startYear": 2018, // Year, "idExpireMonth": "expireMount", "idExpireDays": "expireDays", "idDaysLeft": "daysLeft" });

Full config with localization

$("#epcounter").epcounter({ "amountPromotionDays": 15, "startDay": 1, // Day of month 1 - 31 "startMonth": 8 // Month 1 - 12, "startYear": 2018 // Year, "idExpireMonth": "expireMount", "idExpireDays": "expireDays", "idDaysLeft": "daysLeft", "monthsList": [ "January", "February", "March", "April", "May", "June", "July", "August", "September", "October", "November", "December" ] });

Debug (print to console)

$("#epcounter").epcounter({ "printLog": true // Default: false });
